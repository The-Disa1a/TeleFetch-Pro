## 🚀 Overview
**TeleFetch Pro** is a powerful Chrome Extension designed to bypass Telegram Web's limitations and allow you to effortlessly download Videos, Images, Documents, and Audio files directly to your device.

Say goodbye to paywalls, arbitrary limitations, and usage tracking. This extension was built from the ground up for speed, reliability, and the ultimate user experience.

## ✨ Features

- **🛡️ 100% Free & Unlimited**: No subscriptions, no Kodepay, no download limits. Completely unlocked.
- **📥 Smart Priority Download Queue**: Click download on 50 videos if you want! They will be politely queued up and downloaded automatically, one by one, to ensure maximum network stability without crashing Telegram.
- **⏸️ True Pause & Resume**: Need to halt a massive multi-gigabyte download? Instantly pause the stream and securely archive your bytes in memory. Hit resume, and it jumps back into the queue and continues seamlessly from where it left off!
- **🏷️ Intelligent File Naming**: Tired of messy `9123891283.mp4` filenames? The extension smartly extracts the text caption attached to the media and uses it as the permanent, clean filename. 
- **🧩 Smart Extension Recovery**: Even if your text caption has weird punctuation or dots in it, the native file extensions (`.mp4`, `.jpg`, `.mov`) are automatically calculated and safely preserved.
- **🚫 Built-In Ad Blocker**: Automatically hunts down and completely obliterates standard sponsored Telegram Ads from your chat feed so you can browse peacefully.
- **🌉 Cross-Origin Service Proxy**: Utilizes Chrome's native background worker layer to completely bypass CORS header restrictions, ensuring flawless fetching even for standard images and files.

## 📦 Local Installation

Since this extension avoids the $5 application fee on the Chrome Web Store, you can easily load it directly into your browser!

1. **Download** or clone this repository to your computer.
2. Open Google Chrome and type `chrome://extensions/` into your address bar.
3. Turn on **Developer Mode** using the toggle switch in the top-right corner.
4. Click the **Load unpacked** button in the top-left.
5. Select the unzipped folder containing this extension's files.
6. **Done!** Open Telegram Web, find any media, and enjoy unlimited downloading.
